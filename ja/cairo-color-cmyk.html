Cairo::Color::CMYKクラス

Since 1.5?: CMYK（シアン、マゼンタ、黄色、キー・プレー ト）で色を表現します。

cairoにはないrcairoでの拡張です。

オブジェクト階層

  • Object
    • Cairo::Color::Base
      • Cairo::Color::CMYK

クラスメソッド

Cairo::Color::CMYK.new(c, m, y, k, a=1.0)
CMYKで表現した色を生成します。
  • c: シアン
  • m: マゼンタ
  • y: 黄色
  • k: キー・プレート（黒）
  • a: アルファ値
  • 戻り値: 生成した色

c

  • value: 新しいシアンの値

yellow=(value)

  • value: 新しい黄色の値

  • 戻り値: キー・プレートの値
key_plate=(value)

to_a
to_ary
[cyan, magenta, yellow, key_plate, alpha]を返します。
