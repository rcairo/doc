Cairo::Rectangleクラス

四角を表します。

オブジェクト階層

  • Object
    • Cairo::Rectangle

クラスメソッド

Cairo::Rectangle.new(x, y, width, height)
指定した大きさの四角を作ります。
  • x: 四角の左側のX座標
  • y: 四角の上側のY座標
  • width: 四角の幅
  • height: 四角の高さ
  • 戻り値: 新しく作ったCairo::Rectangleオブジェクト。

インタンスメソッド

height
四角の高さを返します。
  • 戻り値: 高さ
height=(height)
四角の高さを設定します。
  • height: 高さ
width
四角の幅を返します。
  • 戻り値: 幅
width=(width)
四角の幅を設定します。
  • width: 幅
x
四角の左側のX座標を返します。
  • 戻り値: X座標
x=(x)
四角の左側のX座標を設定します。
  • x: X座標
y
四角の左側のY座標を返します。
  • 戻り値: Y座標
y=(y)
四角の左側のY座標を設定します。
  • y: Y座標
